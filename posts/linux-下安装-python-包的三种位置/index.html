Linux 下安装 Python 包的三种位置

在Linux下安装Python包的三种位置：

1.创建虚拟环境，在其中安装包

python3 -m venv env source ./env/bin/activate python3 -m pip install ‘package_name’

2.将包安装在当前用户环境(安装目录一般在用户文件夹下，类似 /home/user/.local/lib/python3.5)

python3 -m pip install –user ‘package_name’

3.强制安装到系统环境(小心炸机^-^)

sudo python3 -m pip install –user ‘package_name’

另，在Debian系列的发行版中，系统会把安装包细分在两个目录下

dist-packages文件夹，存放通过系统的包管理器（i.e. apt install）安装的Python包。

site-packages文件夹，存放通过pip，或从源代码手动构建等其他方式安装的Python包。
